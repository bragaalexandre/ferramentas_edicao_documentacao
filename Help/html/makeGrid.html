Help versão 1.0

Descrição:

O algoritmo gera o vetor de grade em UTM no fuso correto baseado na moldura e reprojeta para o epsg de interesse (Apenas dentro do Brasil)

Parâmetros:

• Camada de moldura → Camada de moldura a ser processada para a gereação do grid sobre toda a sua extensão. • Escala de edição → Escala a ser considerada para a edição. • Gerar cruzetas de lat lon → Gera cruzetas baseadas nas coordenadas de latitude do sistema de referência em selecionado. Atenção: Os elementos da moldura devem estar todos no mesmo fuso-horário.

Exemplo de uso:

• Gerar cruzetas de lat lon → ativado

Exemplo de Saída:

A grade é gerada sobre cada feição da moldura, ainda que se trate de uma moldura gerada a partir de multipolígonos não interligados entre si.
