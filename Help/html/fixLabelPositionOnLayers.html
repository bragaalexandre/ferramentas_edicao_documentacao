Help versão 1.0

Descrição:

Fixa posição dos labels de acordo com o calculado pelo QGIS.

Parâmetros:

• Selecionar camadas → Camadas sobre as quais o processing vai atuar. • Limite geográfico → Camada do tipo polygon que limitará a espacialmente a atuação do processing. • Escala → Escala correspondente ao processing. • Substituir rótulos já fixados → Uma vez assinalada esta opção, os rótulos já fixados serão substituídos pelos rótulos determinados a partir do processing. Para não modificar os rótulos já fixados, deixe esta opção desmarcada.
